# lebai_vision_grasp 操作说明

Lebai LM3 机械臂 + 平行夹爪的视觉抓取仿真包。支持 MoveIt 2 fake hardware 仿真和真机两种模式。

---

## 目录结构

```
lebai_vision_grasp/
├── config/
│   ├── vision_grasp_sim_params.yaml   # 仿真模式参数
│   └── vision_grasp_params.yaml       # 真机模式参数
├── launch/
│   ├── vision_grasp_sim.launch.py     # 仿真一键启动
│   └── vision_grasp_real.launch.py    # 真机启动
└── lebai_vision_grasp/
    ├── vision_grasp_orchestrator.py   # 抓取主控节点
    ├── fake_target_position.py        # 仿真目标发布节点
    └── gripper_sim_node.py            # 仿真夹爪桥接节点
```

## 仿真模式

### 环境要求

- ROS 2 Humble
- MoveIt 2
- 已编译 `lebai_with_gripper_and_shaft_moveit_config` 包

### 编译

```bash
cd ~/lebai_ws
colcon build --packages-select lebai_vision_grasp --symlink-install
source install/setup.bash
```

### 启动方式一：一键启动（推荐）

```bash
cd ~/lebai_ws && source install/setup.bash
ros2 launch lebai_vision_grasp vision_grasp_sim.launch.py
```

启动后会自动拉起：MoveIt + RViz、假目标发布节点、抓取编排节点。

### 启动方式二：分终端启动（便于调试）

**终端 1 — MoveIt + RViz**
```bash
cd ~/lebai_ws && source install/setup.bash
ros2 launch lebai_with_gripper_and_shaft_moveit_config demo.launch.py
```

**终端 2 — 假目标发布节点**
```bash
cd ~/lebai_ws && source install/setup.bash
ros2 run lebai_vision_grasp fake_target_position \
  --ros-args --params-file \
  /home/zhengshaolong/lebai_ws/src/lebai_vision_grasp/config/vision_grasp_sim_params.yaml
```

**终端 3 — 抓取编排节点**
```bash
cd ~/lebai_ws && source install/setup.bash
ros2 run lebai_vision_grasp vision_grasp_orchestrator \
  --ros-args --params-file \
  /home/zhengshaolong/lebai_ws/src/lebai_vision_grasp/config/vision_grasp_sim_params.yaml
```

### 触发抓取

等待终端 1 出现 `You can start planning now!` 后，在新终端执行：

```bash
cd ~/lebai_ws && source install/setup.bash
ros2 service call /vision_grasp_orchestrator/execute_once std_srvs/srv/Trigger {}
```

### 抓取流程说明

1. 夹爪张开
2. 机械臂规划并运动到接近点（目标正上方）
3. 笛卡尔直线下降到抓取点
4. 夹爪闭合（保留间隙，不穿透方块）
5. 方块 marker 附着到 tool0 坐标系，跟随末端运动
6. 笛卡尔直线提升

### RViz 配置

在 RViz 中添加以下显示项：

| 类型 | Topic | 说明 |
|------|-------|------|
| Marker | `/vision_grasp_orchestrator/markers` | 目标方块、抓取点、接近点、提升点 |
| RobotModel | — | 机械臂模型（默认已有） |
| PlanningScene | — | 桌面碰撞体（默认已有） |

## 关键参数说明

参数文件：`config/vision_grasp_sim_params.yaml`

### 目标位置（fake_target_position 节点）

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `fixed_x_m` | 0.35 | 方块 X 坐标（米），LM3 可达范围约 0.2~0.5m |
| `fixed_y_m` | 0.00 | 方块 Y 坐标，0 = 正前方 |
| `fixed_z_m` | 0.11 | 方块中心 Z 坐标，需高于桌面顶面 |
| `motion_mode` | fixed | `fixed` = 固定位置，`sweep` = 椭圆扫描 |

### 抓取偏置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `grasp_offset_z_m` | 0.172 | tool0 到指尖中心距离（SolidWorks 实测，单位米） |
| `approach_height_m` | 0.12 | 接近点在抓取点正上方的高度 |
| `lift_height_m` | 0.14 | 抓取后提升高度 |

### 夹爪参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `gripper_open_rad` | 1.0 | 张开角度（弧度），对应 URDF 上限 |
| `gripper_closed_rad` | 0.5 | 抓取角度；0.0 = 完全闭合（指尖间距约 2.3cm），4cm 方块用 0.5 |

### 桌面碰撞体

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `table_pose_x_m` | 0.35 | 桌面中心 X |
| `table_pose_z_m` | 0.04 | 桌面中心 Z，顶面 = `pose_z + size_z/2` = 0.09m |
| `table_size_z_m` | 0.10 | 桌面厚度 |

### MoveIt 规划参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `allowed_planning_time` | 15.0 | 单次规划超时（秒），越大成功率越高 |
| `num_planning_attempts` | 30 | 规划重试次数 |
| `planner_id` | RRTConnectkConfigDefault | 规划器，可选 `RRTstarkConfigDefault`、`ESTkConfigDefault` |

## 手动控制夹爪

```bash
# 张开夹爪
ros2 service call /vision_grasp_orchestrator/open_gripper std_srvs/srv/Trigger {}

# 闭合夹爪
ros2 service call /vision_grasp_orchestrator/close_gripper std_srvs/srv/Trigger {}
```

## 常见问题

**规划失败 `error_code=99999`**
- 增大 `allowed_planning_time` 或 `num_planning_attempts`
- 检查目标位置是否在机械臂可达范围内（LM3 最大臂展约 0.54m）

**IK 求解失败 `error_code=-31`**
- 目标位置超出工作空间，调整 `fixed_x_m` / `fixed_z_m`

**笛卡尔路径不完整 `fraction < 0.90`**
- 降低 `cartesian_fraction_threshold`（如改为 0.80）
- 检查桌面碰撞体是否阻挡路径，调整 `table_pose_z_m`

**方块不在桌面上**
- 调整 `fixed_z_m`，使其等于桌面顶面 Z + 方块半高
- 桌面顶面 Z = `table_pose_z_m` + `table_size_z_m / 2`
- 例：`table_pose_z_m=0.04`，`table_size_z_m=0.10` → 顶面 Z=0.09，方块中心 Z=0.09+0.02=0.11

## 真机模式

> 真机模式需要机械臂控制器已启动并可访问。

```bash
cd ~/lebai_ws && source install/setup.bash
ros2 launch lebai_vision_grasp vision_grasp_real.launch.py \
  robot_ip:=10.20.17.1 \
  start_robot_stack:=true
```

真机参数文件为 `config/vision_grasp_params.yaml`，需将 `gripper_action_name` 改为真机夹爪控制器接口。
